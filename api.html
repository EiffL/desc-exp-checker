API

The API allows you to query our submission database for a list of the categorized problems. You'll get them anonymized and without a timestamp, the format is JSON, which should be digestible in almost any programming language. If you need at different format, open a github issue.

Any other queries?

The set of queries we currently allow is restricted to global queries based on the problem class. If you want to be able to quiery our database differently, tell us by opening a github issue.

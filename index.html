DES exposure checker

See how DES images actually look like. Discover flaws we would otherwise have missed. Make our data better.

Introduction

The purpose of this website is to provide easy access to the latest DES SV-A1 finalcut images. Well, not exactly the finalcut images. What you see here are factor 4 downsampled versions of the actual images just to keep the loading times and image sizes tolerable.

What's the goal of this?

We all want to conduct a science analysis based on the DES data. However, no data is perfect, so we need to know what kinds of image artifacts are in our data because, ultimatively, all of them need to be accounted for in one way or another. This application helps in two ways:

  • It gives a convenient visual impression of the actual DES data and of the processing steps that are already in place to identify and correct for or mask artifacts.
  • It enables everyone to search for undetected artifacts. We gather your submissions, analyze them, and feed them back to the science working groups and DESDM. This way, we can train the processing pipelines on flaws, of which we know they are there.

How can I start?

Check out our Tutorial and you'll know what to do in no time.

How are we doing so far?

The basic statistics are computed in real time. More detailed analyses will become available once we have gathered enough submissions, but you can go to our API page and download the anonymized submissions right now.

How can I give feedback?

The code is on github. Please open a new Issue to let us know what we could add, improve, ...

If you find particular failure cases and you think they may constitute a new class of artifacts, please go to the DESDM users wiki, where we list and discuss such cases.

Statistics

The first pie chart shows how many images have been checked so far, in relation to the entire set of images we have in our database. The second chart shows how many of the checked images have been found Fine or Problematic.

Below you find a break-up of the images that have been found problematic by at least one person. The reason each of these problem classes are shown independently is that there can be multiple problems in any given image, so that the sum of all of these problems can be larger than the sum of all checked files.
